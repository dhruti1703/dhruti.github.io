About Me

I am a certified SAP Integration Consultant with 4+ years of experience in delivering scalable, secure, and performance-oriented solutions using SAP Cloud Platform Integration (CPI). My core expertise lies in integrating SAP SuccessFactors, S/4HANA, and various third-party systems (SAP and non-SAP) with APIs, SFTP, and real-time communication. I have delivered high-impact integration projects for global clients including British American Tobacco (BAT), RAI, ITC Infotech, TCS, Capgemini, and Volvo.

Expertise

  • End-to-End SAP CPI Development (iFlows, Groovy Scripts, OData, SOAP, REST, SFTP)
  • SuccessFactors Integration (Employee Central, OData, CompoundEmployee, Integration Center)
  • S/4HANA Integration (On-Premise and Cloud)
  • API Management in SAP BTP
  • Boomi to CPI migration & Cloud Integration Best Practices

Major Clients & Projects

  • British American Tobacco (BAT) via ITC Infotech:
    Led the migration of legacy Boomi-based integrations to SAP CPI, primarily focused on SuccessFactors Employee Central integration. Designed and implemented secure data flows using SFTP and PGP encryption for employee master data. Built custom iFlows involving Groovy scripting, dynamic routing, and error handling. Ensured data consistency and security during payroll data transfers and master data synchronization between SuccessFactors and SAP S/4HANA.

  • Volvo via Capgemini:
    Designed and implemented SAP CPI integrations for HR processes between SAP SuccessFactors and On-Premise SAP S/4HANA. Handled value mappings, lookup tables, local process integrations, and complex transformations using Groovy scripts. Developed real-time OData and SOAP-based integrations for onboarding, employee updates, and org data replication.

  • RAI (Retailers Association of India):
    Built multi-tenant CPI interfaces to manage retail workforce integration between SuccessFactors and third-party payroll providers. Developed modular integration flows for multiple vendors with token-based authentication and error alerts.

  • ITC Infotech Internal Project:
    Created a pre-configured CPI integration template to connect SAP SuccessFactors with Microsoft Azure AD and SAP ECC systems. Delivered a plug-and-play architecture for small retail clients using message mapping and secure token-based APIs.

  • TCS:
    Collaborated on SuccessFactors and SAP HCM hybrid model where CPI was used to bridge cloud and on-premise systems. Developed monitoring dashboards and proactive error-handling alerts using SAP Cloud Integration content modifier and mail adapters.
